Forensics Challenge Solution

SOLUTION

Background knowledge: In forensics, a lot of steganography is used. Steganography is basically hiding a message/file within another file (most commonly an image).

There are many useful tools in steganography, but a particular tool called "Steghide" is very prominent in CTFs. Steghide takes advantage of images having many bits, so many that if we change a few there will be no noticable changes when you look at the image. By changing the bits in the image, we can hide messages and files within it.

Download Steghide

Step1: Try to uncover the hidden message in the image using steghide.

There is only 1 image on this page so it's a good idea to start with that one! Steghide is easiest to use in linux so that's what I will be using. If you look at the list for steghide commands, you will easily find the command to extract a file from another, which is the one we want to use. This is the command that should be used:

steghide extract -sf smudge1.jpg

After you enter this command, Steghide will ask you to enter the passpharase.

Step2:The passphrase

Many times, no passpharase is set and you can just hit enter. This one does have a passpharese though. How do we find what it is? This is where the hint comes in handy, note that it says that Smudge's fried has the *key* aka password!

When we go to the new page, we see that there are a lot of references to Mike's mic brand. This is because that's were the passpharase is hidden. This is a way easier method to hide a message in a picture, simply by making the message some-what un-noticable unless given the right clues! When we look at the image closely and zoom in, we can make out a potential passphrase.

MEOW!!!!!!!!!!!1 can be seen on the mic.

Step3:Retrieving the hidden file

If you enter the correct passpharase (which is 'MEOW!!!!!!!!!!!1' in this case), steghide will extract the hidden textfile from the image. What you're left with is a file that contains the message: "Dear smudge, you're looking rather dapper today. I am sending you 10 pieces of tuna. Enjoy!"

Step4:Finally solving the challenge

If you send Smudge the message uncovered with steghide, it will solve the challenge. Smudge is happy again! Good job!
